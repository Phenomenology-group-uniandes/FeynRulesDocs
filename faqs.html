Connecting to FeynArts

Last updated: Nov 12th, 2020

FeynArts

Even though, FeynRules provides a wide set of options for model validation, the FeynArts software can be more convenient for that purpose, given that it provides a way to draw Feynman diagrams and compute the Feynman Amplitude. FeynArts allows to compare the computed results with the analytical values derived from the model. As FeynArts is another Mathematica package, the user can check the model using the same notebook for model generation. Thus, it reduces the amount of time needed to validate the computational model.

Key Specs

The basic level tools for FeynArts are summarized in chapters 3, 4 and 5 of the user's manual. Moreover the set of options for model validation in FeynRules can be found in chapter 3,4 and 8 of the user's manual

To export the model to FeynArts the user can run the following Code.

FeynArts exportation example
LRen = OnShellRenormalization[LSM + LN, QCDOnly -> True, FlavorMixing -> False, Exclude4ScalarsCT -> True];
SetDirectory["/FeynArts-3.11/Models"]
WriteFeynArtsOutput[LRen, GenericFile -> False, FlavorExpand -> True, Output -> "HeavyN_FA"];
